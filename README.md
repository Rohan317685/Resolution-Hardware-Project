# Resolution-Hardware-Project

This is a falstad simulator RC circuit, it basically is just a pattern that switches of leds 1 by 1 originally it was meant to be a sort of timer however due to it being hard to predict the time i decided to go for a pattern instead.  It is also my first time using it so it is quite new to me!

It has a 9V power source, 3 leds adn 3 resistors and 3 capacitors and 1 switch.

It works using the RC circuit!it has the capacitor slowly leak out power when the switch is off to time it sort of accurately however it seems to be quite difficult for me!
