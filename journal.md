# Led pattern

I'm building a led pattern because I want a sort of leds switching off.At the start the original plan was to have a timer sort of like a rocket launch timer. But the timing 
was REALLY difficult so i decided to go for a pattern where the leds switched off 1 by 1 instead of the same time.

## Date - 16March

Today I worked on making the entire schematic and research RC circuits.

![image](https://cdn.hackclub.com/019cf48a-7eb3-715d-9731-89dcc12014c1/Screenshot%202026-03-16%2002.46.50.png)
